### Getting Started


#### .env file

```
AWS_ACCESS_KEY_ID=<your-access-key>
AWS_SECRET_ACCESS_KEY=<your-secret-access-key>
```

```shell script
$ sudo npm i -g serverless
$ npm i
# test locally
$ npm run local
# deploy
$ npm run deploy
```
